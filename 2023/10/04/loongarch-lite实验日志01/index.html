目标:完善TEMU的基本功能

TEMU的阅读和修改

TEMU简介

TEMU是一个轻量级的动态二进制仿真器,能够仿真对应平台的二进制机器码(应该是在原天大小学期程设实践课程的部分,因故移动至此).
其基本功能和调试器类似,需要实现类似单步执行,设置断点之类的功能.

代码框架

TEMU针对计算机中不同的硬件部分分别实现了接口,其基本结构如下:

int main(int argc, char *argv[]) {
//带参数的监视器初始化
init_monitor(argc, argv);

//模拟启动计算机系统
restart();

//循环取指的UI界面
ui_mainloop();

return 0;

}

static int cmd_c(char *args) {
cpu_exec(-1);
return 0;
}

2*(3+4)-5

2 参与计算的数字

  • 运算符
    ( 运算符
    3 参与计算的数字
  • 运算符
    4 参与计算的数字
    ) 运算符
  • 运算符
    5 参与计算的数字
我们所拆出的每个元素就可以称之为一个 `token` ,每个token有不同的类型,对不同类型的token进行不同规则的运算,这就是词法分析的基本原理.

TEMU中使用正则表达式来完成对token的匹配,将一段字符串解析成各个token,然后按照对应的规则进行计算,计算部分将在下个日志完成.
